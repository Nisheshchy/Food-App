

# Food App
A simple, modern, and fully responsive restaurant landing page built from scratch using React.js, Material UI (MUI), and React Icons.

# Features
- Modern UI/UX: Designed based on a professional Figma template.

- Responsive Design: Fully optimized for desktops, tablets, and mobile devices.

- Component-Based Architecture: Modular structure including Home, About, Work, Testimonials, Contact, and Footer sections.

- Interactive Elements: Includes a functional mobile drawer menu using MUI.

## Tech Stack
- Framework: React.js

- Styling: CSS3 (Custom styles provided in App.css)

- Icons: React Icons

- UI Components: Material UI (MUI)

- Fonts: Reem Kufi via Google Fonts



## Installation & Setup
Clone the repository:

``` Bash

git clone https://github.com/Nisheshchy/Food-App.git
```
Install Dependencies:

``` Bash

npm install
```
### Install specific libraries 
``` Bash
npm install react-icons @mui/material @emotion/react @emotion/styled @mui/icons-
```
Start the Development Server:

```Bash

npm start
```
